ARRAYS


An array is a collection of similar data elements stored at contiguous memory locations. It is the simplest data structure where each data element can be accessed directly by only using its index number.
The first element of the array refers to index 0.
1D Array:

Syntax : int arr[max_columns];
The address of any element of a 1D array can be calculated by using the below-given formula:
Byte address of an element A[i] = base address + size * (i â€“ first index)

2D Array:
2 Dimensional arrays are often defined as an array of arrays. A 2D array is also called a matrix. A matrix can be depicted as a table of rows and columns.
Syntax : int arr[max_rows][max_columns];

Row Major Order: Address(a[i][j]) = B. A. + (i * n + j) * size
Column Major Order: Address(a[i][j]) = ((j*m)+i)*Size + BA
m is the number of rows & n is the number of columns

Operations on Array


1. Insertion in Array:

Pseudo Code:
arr[ 2 ] = 10;
Time Complexity:
O(1) to insert a single element O(N) to insert all the array elements [where N is the size of the array]

2. Access elements in Array:
Pseudo Code:
return arr[ 2 ] ;
Time Complexity:
O(1)

3. Searching in Array:
Pseudo Code:
Loop from i = 0 to 5:
check if arr[i] = 2:
return true;
Time Complexity:
O(N), where N is the size of the array.
